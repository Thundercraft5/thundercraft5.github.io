## <DevWikiIcon size={24} /> FANDOM Developers (2019-2022)
### <JavaScriptIcon size={24}/> JavaScript
#### Scripts That I created
- [MoreDropdown](https://dev.fandom.com/wiki/MoreDropdown) - Adds a useful dropdown to the page edit button that contains some tools.
- [BlockListTools](https://dev.fandom.com/wiki/BlockListTools) - Adds some links to the blocklist and makes it cleaner
- [BlockLookup](https://dev.fandom.com/wiki/BlockLookup) - A tool for managing and viewing blocked users on your wiki in a clean format
- [AjaxAbuseLog](https://dev.fandom.com/wiki/AjaxAbuseLog) - A modal script that makes browsing abuselogs easier
- [AjaxContentModel](https://dev.fandom.com/wiki/AjaxContentModel) - A modal script that makes changing content models easier
- [AjaxRename](https://dev.fandom.com/wiki/AjaxRename) - A modal script that makes renaming pages a breeze
- [QuickPurge](https://dev.fandom.com/wiki/QuickPurge) - A script that allows you to skip confirmations for purging pages
- [GlobalJSLinks](https://dev.fandom.com/wiki/GlobalJSLinks) - A script that allows you to view a list of your global JS
- [CollapseGlobalNavButton](https://dev.fandom.com/wiki/CollapseGlobalNavButton) - A script that allows you to collapse the global navigation
- [TitleBlacklistHighlight](https://dev.fandom.com/wiki/TitleBlacklistHighlight) - A script that highlights the syntax of MediaWiki:Titleblacklist to make it more readable
- [Reconstitution](https://dev.fandom.com/wiki/Reconstitution) - A script that mass restores deleted pages for a specific user or all users
- [RevisionDelete](https://dev.fandom.com/wiki/RevisionDelete) - A script that allows users to selectively remove revisions from page histories
- [PageReport](https://dev.fandom.com/wiki/PageReport) - A script that provides a detailed parser report on a page and locates hidden lua errors.

#### Scripts That I maintained
- [Nuke](https://dev.fandom.com/wiki/Nuke) - A script to mass delete pages
- [UserAndIPTools](https://dev.fandom.com/wiki/UserAndIPTools) - A script to view the information of users/IPs and provides tools for IPs
- [MultipleFileDelete](https://dev.fandom.com/wiki/MultipleFileDelete) - A script to selectively delete pages on certain special pages
- [SnowStorm](https://dev.fandom.com/wiki/SnowStorm) - A script to add animated snow to a wiki
